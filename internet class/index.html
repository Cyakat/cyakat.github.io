How Web Applications Work

A web application is software that is provided through your browser. It is what is known as Software as a Service. Web applications are becoming more popular because of their ease of use for the consumer. Web applications do not need to be installed on every machine you want to use them on and some are also available to use on your smartphone. Common web applications are sites such as one drive, google, drive, Word (on your browser), and Google docs. Read more here.

Progressive Web Application

Progressive web applications or PWAs are web applications that can be downloaded onto a machine. It still functions the same as it would on your browser but with more functionality and reduced loading time and increased convenience. PWAs benefit from being a web app and an installed app. Since the application is still web based, it allows seamless switching from the installed version to the browser version. Many companies are switching to this approach and a well known company that is already making use of this technology is Discord. They have an installable app for computers and phones but it is also available on the web so you are still able to access the application in case you are on a new device. Read more here.
