Minecraft Server Bot

The Minecraft Server Bot is what I created initially to make it easier to get information about my Minecraft servers. The bot allows users to look up an ip for each server as well as restart the server if it ever is down.

I made this bot because my modded Minecraft servers kept crashing (no surprise there) and I would have to be notified if they went down so I could restart them from my phone. This became a bit cumbersome and annoying after a while for me and my friends. I then created this discord bot that would allow them to restart the server if it went down in most cases.

However, I am not the greatest at design and so my friends found it confusing to use the bot for anything except restarting. They often forgot about its ability to give IP addresses for the minecraft servers.

This took a few redesigns of the /ip command and I think that it is adequate now.
