All User message data is not stored in any capacity; messages are read and scanned for certain key words set by a guild's owner or its moderators. User ID's are stored for identification and synchronization purposes. By using this bot users agree to allow other users to view their discord name and tag, regardless if said users share any server with one another. All users who have shared a guild with Word Tracker at any point in time permit storage of their user ID. All messages which are readable to Word Tracker are scanned to obtain usage information. Users can request this data to be deleted and can opt out of tracking.
